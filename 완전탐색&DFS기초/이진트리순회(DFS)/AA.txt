def DFS(v):
    if v>7:
        return
    else:  
        print(v, end=' ')
        DFS(v*2)
        DFS(v*2+1)
        
if __name__=="__main__":
    DFS(1)

'''
트리 구조로 각 요소를 노드라 하고,
최상위를 루트 노드라 한다.
DFS는 순서가 기준 부모 노드에서 왼쪽 아래부터 탐색해서
탐색이 끝나면 오른쪽 위로 탐색한다.
세 가지 순회가 있는데 전위,중위,후위 순회 방식이다.
전위는 부모,왼쪽(자식),오른쪽(자식) 순이고,
중위는 왼쪽,부모,오른쪽 이며,
후위는 왼쪽,오른쪽,부모이다.
해당 코드는 전위 순회 방식이다. 즉, print문이 부모 노드로
print위치로 순회 방식을 설정할 수 있다.
'''
